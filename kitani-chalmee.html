Kitani Chalmee

Branding / Web Design

Kitani Chalmee is a Japanese physiotherapist who works with a number of Premier League players.

The client wanted a clear hierarchy between the characters, focussing more on his first name.

Exploring different ideas from human anatomy to shapes creating harmony.

A simple yet elegant logo design that maintains a clear hierarchy throughout. The idea is based on shapes supporting each other and creating a harmonious monogram.

Desinged the website using Adobe XD and built using Webflow.
